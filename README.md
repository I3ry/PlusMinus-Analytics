# PlusMinus Analytics

A basketball performance intelligence platform that helps players track performance, receive AI-driven insights, and follow personalized training plans.

## Features

- **Player Dashboard**: Real-time performance metrics and ratings
- **Shot Analytics**: Visual shot charts and heatmaps
- **AI Analysis**: Computer vision-powered performance insights
- **Training Plans**: Personalized workout recommendations
- **Progress Tracking**: Long-term improvement analytics

## Tech Stack

- **Frontend**: React.js with Hooks
- **Backend Simulation**: JSON Server
- **Data Visualization**: Custom React components
- **AI Simulation**: Mock analysis engine
